# Yelp data

Please download Yelp dataset from https://www.kaggle.com/yelp-dataset/yelp-dataset and unzip it here!

You must have the following CSV files:

* yelp_business.csv
* yelp_business_attributes.csv
* yelp_business_hours.csv
* yelp_checkin.csv
* yelp_review.csv
* yelp_tip.csv
* yelp_user.csv
